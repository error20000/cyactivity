## 1、查询配置信息


### 1.1、查询奖励档位


* **请求地址**

> /wb/tpl0612/gift/type/all

* **请求方式**

> get

* **请求参数**



* **返回字段**

|参数|描述|
|:------|:------|
|pid|主键|
|name|奖励档位的名称|
|num|奖励达成人数|
|app_code|应用代码|
|activity_code|活动代码|
|version_code)|活动版本|


* **接口示例**

```javascript

 	"code": 20000, // 20000 表示成功

 }
 
```

### 2.2、查询邀请总人数

> 根据邀请码，查询邀请总人数

* **请求地址**

> /wb/tpl0612/active/invite/num/{inviteCode}

* **请求方式**

> get

* **请求参数**

|参数|必选|类型|描述|
|:------|:------|:------|:------|
|inviteCode|是|String|邀请码|

* **返回字段**

|参数|描述|
|:------|:------|
|data|邀请人数|


* **接口示例**


```javascript

 {
 	"code": 20000,
 	"msg": "成功",
 	"data": 1
 }
 
```
### 2.3、通过手机号查询预约信息


* **请求地址**

> /wb/tpl0612/active/phone/{phone}

* **请求方式**

> get

* **请求参数**

|参数|必选|类型|描述|
|:------|:------|:------|:------|
|phone|是|String|手机号|

* **返回字段**

|参数|描述|
|:------|:------|
|pid|主键|
|app_code|应用代码|
|openid|openid|
|create_time|参加时间|
|plat|预约平台|
|phone|预约手机号|
|invite_code|获得的邀请码|
|used_code|使用的邀请码|
|activity_code|活动代码|
|version_code)|活动版本|


* **接口示例**


```javascript

 {
 	"code": 20000,
 	"msg": "成功",
 	"data": {
 		"pid":"",
 		"app_code":"",
 		"openid":"",
 		"create_time":"",
 		"plat":"",
 		"phone":"",
 		"invite_code":"",
 		"used_code":"",
 		"activity_code":"",
 		"version_code":"",
 	}
 }
 
```

## 错误码说明


|错误码|描述|
|:------|:------|
|20000|成功 |
|20001|失败 |
|20002|保存失败|
|20003|修改失败|
|20010|手机号不合法|
|20011|已预约|
|20012|邀请码已用完|
|20013|邀请码获取失败，请重新提交|
|20014|手机验证码错误|
|20020|缺少参数|
